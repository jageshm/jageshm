h1. Anyway Feedback

Simple WordPress plugin to gather user feedback.

h2. Install

# Upload anyway-feedback folder to wp-content/plugins
# Enable it from plugin page of admin-panel.
# Setting up at Options > Anyway Feedback under instruction.
# That's all.

h2. Usage

Typical usage is FAQ. You can display without any coding. Almost everything can be set up at admin panel.

You can also customize by template tags. Manual is on admin panel.
